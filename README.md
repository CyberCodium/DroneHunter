## Features

- **Real-time RF signal analysis** and drone fingerprinting
- **Automatic threat classification** with configurable thresholds
- **Acoustic + visual alerts** for detected threats
- **5-tab professional dark-theme GUI** (PyQt5)
- **210+ known drone RF signatures** in the classification database
- **Spectrum waterfall**, signal strength history, alert log
